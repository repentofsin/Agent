## First Time Setup (5 minutes)

### Step 1: Install Node.js
If you don't have Node.js:
1. Go to https://nodejs.org
2. Download the LTS version (recommended)
3. Install it (just click Next, Next, Next...)

### Step 2: Start the Server

**Windows Users:**
- Double-click `START.bat`

**Mac/Linux Users:**
- Double-click `START.sh`
- OR open Terminal and run: `./START.sh`

**OR Manual Method (All Platforms):**
```bash
npm install
npm start
```

### Step 3: Open Browser
- Go to http://localhost:3000
- The app will load automatically

### Step 4: Get API Keys

**Anthropic (Claude AI):**
1. Go to https://console.anthropic.com
2. Sign up (free!)
3. Get $5 in free credits
4. Copy your API key (starts with "sk-ant-")

**ElevenLabs (Voice):**
1. Go to https://elevenlabs.io
2. Sign up (free tier available)
3. Copy your API key

### Step 5: Start Practicing!
1. Paste both API keys into the app
2. Choose a scenario (FSBO, Expired Listing, etc.)
3. Click "Start Practice Session"
4. Talk when prompted!
